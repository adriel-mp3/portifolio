# Portfólio

Seja bem-vindo(a) ao meu portfólio. O objetivo dele é apresentar as principais informações da minha carreira como desenvolvedor front-end de forma simples e didática.

## Visão Geral

Meu portfólio é dividido em três seções: uma breve descrição contendo meu nome e curso, meus projetos, informações sobre mim e uma seção de contato. Destaquei a seção de projetos com um background chamativo e um slide 100% responsivo, proporcionando uma experiência agradável em diferentes dispositivos. Na seção "Sobre", utilizei listas de conteúdo para tornar o texto mais atrativo e de fácil leitura. Na parte de contato, disponibilizei um meio direto de envio de e-mail utilizando atributos HTML.

## Tecnologias Utilizadas
- HTML & CSS
- JavaScript
- Figma

Quanto às tecnologias utilizadas, acredito que a tríade mencionada seja essencial. Por ser uma página única, com o objetivo de destacar meus projetos e proporcionar uma experiência agradável para meu público-alvo.

## Conclusão

Estou bastante satisfeito com o resultado. Acredito que todo projeto pode ser sempre aprimorado, e estarei sempre disposto a atualizar este portfólio sem perder sua essência, que é a simplicidade na transmissão das informações relevantes para o usuário final.
